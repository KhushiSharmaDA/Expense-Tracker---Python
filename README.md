# Expense Tracker (Python)

A Python-based expense tracker project that analyzes personal spending data using pandas, numpy, and matplotlib.

## Features
- Data cleaning and processing using pandas
- Numerical analysis using numpy
- 6 visualizations created with matplotlib
- Monthly expense reports exported as CSV files

## Project Structure
- data/ : Raw and processed datasets
- reports/ : Monthly and summary reports
- visuals/ : Saved charts and graphs
- scripts/ : Python scripts for processing, analysis, and visualization

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
